# Plateforme des artisans

Projet de site web réalisé dans le cadre d’un devoir de formation.
La plateforme permet de mettre en relation des particuliers avec des artisans
via un système de recherche par catégorie et des fiches artisan détaillées.

---

##  Fonctionnalités
- Consultation des artisans par catégorie
- Recherche d’un artisan
- Fiche artisan avec formulaire de contact
- Candidature pour devenir artisan
- Messagerie simulée
- Pages légales (mentions légales, données personnelles, cookies, accessibilité)
- Site responsive (mobile / desktop)

---

##  Technologies utilisées
- React
- Vite
- JavaScript (ES6)
- React Router
- SCSS
- Bootstrap
- JSON (données simulées)
- GitHub Pages (déploiement)
